# AI 审讯室

一个可直接打开试玩、也可一键部署到 Vercel 的网页 MVP。

## 项目简介

这是一个 AI 互动剧情解谜游戏原型。玩家通过与 3 名嫌疑人对话、追问时间线和线索，最终完成指认与结案。

## 当前玩法

- 1 个完整案件
- 3 名嫌疑人
- 自由输入问题
- 关键词驱动的角色回应
- 自动解锁线索板
- 最终指认与结局反馈

## 本地打开

直接双击 `index.html` 即可。

如果你想本地起服务，也可以执行：

```bash
python3 -m http.server 4173
```

然后访问 `http://127.0.0.1:4173`。

## 部署

这是纯静态站点，不需要构建步骤。

- GitHub：直接上传整个项目目录
- Vercel：导入仓库后选择默认静态部署即可

项目已包含 `vercel.json`，适合直接发布。
